## What is Cerul?

**Cerul is the video search layer for AI agents.** It lets agents and
developers search video by meaning — across speech, visuals, and on-screen
text — through a single API.

## Short description (≤280 chars)
> Cerul gives AI agents a Search API for video. Query by meaning and get back the exact moments across speech, visuals, and on-screen text — from tech talks, podcasts, conferences, and earnings calls.

## Long description
> Cerul is a video understanding search API built for AI agents. Instead of
> scrubbing through hours of video, agents can ask Cerul a natural-language
> question and get back the precise moments that answer it — with timestamps,
> transcripts, and frame references. Cerul indexes speech, visuals, and
> on-screen text together so a single query can find "what was said", "what was
> shown", and "what was presented" across tech talks, podcasts, conference
> presentations, and earnings calls.
